# Laravel - API REST - SWAGGER
* Simple example of a API REST with Laravel and swagger.

* A simple crud PRODUCTS.

## Installation


### Clone the repository
```
git clone https://github.com/OtavioBernardes/laravel_api_rest_swagger
```

### Then do a composer install
``` 
Composer install 
```

### Set the application key
``` 
php artisan key:generate
```
<br/>

### Add `L5_SWAGGER_GENERATE_ALWAYS=true` in `.env`.
<br/>

### Run migrations and seeds
``` 
php artisan migrate
php artisan db:seed
```

### Acess `app/api/docs`
