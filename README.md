# laravel_api_rest_swagger
Simple example of a API REST with Laravel and swagger

## Instalation

1 - Run composer install

2 - Add `L5_SWAGGER_GENERATE_ALWAYS=true` in `.env`.

3 - Run migrate and seed.

4 - Acess `app/api/docs`
